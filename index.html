01
Foundations — HTML, CSS, JavaScript, Git
Month 1 · Weeks 1–4 · 10 hrs/week minimum
START HERE
▼
Why this first: Life OS is pure HTML/CSS/JS. You can edit it right now. Understanding it deeply lets you host your own dashboard within weeks — the fastest win.
Week 1 — How the web works + HTML + Git
  • Day 1 — Install VS Code (code.visualstudio.com, free). Open your Life OS HTML in it. Read every line — don't change anything yet.
  • Day 1 — Create a GitHub account (github.com). Do the Odin Project "Introduction to Git" lesson. Follow every step.
  • Days 2–7 — The Odin Project Foundations (theodinproject.com). Start lesson 1. Do NOT skip anything including setup.
  • Watch CS50x 2026 Week 0 (cs50.harvard.edu/x, free). Best lecture on how computers work before you touch code.
Week 2 — CSS (Flexbox + Grid)
  • Continue Odin Foundations through CSS. Flexbox and Grid are the most important lessons. After each lesson, open Life OS and find where you're using it. Change it. Break it. Fix it.
  • Watch CS50x Week 8 (HTML/CSS/JavaScript lecture). Clearest explanation of how web pages actually work.
Weeks 3–4 — JavaScript
  • Continue Odin JS sections — do every lesson and exercise in order.
  • Download free: Eloquent JavaScript 4th Edition — eloquentjavascript.net — PDF or epub. Read chapters 1–6 alongside Odin. Do every exercise.
  • Use as reference: javascript.info — Part 1, chapters 1–9. Look things up here when Odin or Eloquent JS confuses you.
  • Week 4 milestone: Create free Netlify account (netlify.com). Drag Life OS HTML file in. Get a real URL. 10 minutes. You now have a live product.
Month 1 milestone: Life OS hosted at a real URL. You can edit HTML in VS Code, push to GitHub, Netlify auto-updates. That's a real developer workflow.
02
Life OS Goes Live — Supabase + Plaid + Real Data
Month 2 · Weeks 5–8
MONTH 2
▼
The shift: Life OS saves to localStorage — dies when you clear the browser. This month you connect to Supabase (real database) and Plaid (live bank). Finance tab updates automatically.
  • Week 5 — Supabase: supabase.com — free account. Read "JavaScript quickstart" (20 min). First task: move wiki from localStorage to a Supabase table. ~40 lines of code. Use Claude to help write it.
  • Week 6 — Async JS + Anthropic API: javascript.info Part 1 Chapter 11 — Promises and async/await. Then docs.anthropic.com "Messages API" quickstart. Call Claude from your own code for the first time.
  • Week 7 — Plaid: plaid.com — free developer sandbox. Follow quickstart. Wire balance into Finance tab. The number updates live.
  • Week 8 — Domain: namecheap.com — ~$12/year. Connect to Netlify in one click. Life OS at your real URL.
  • Download free: You Don't Know JS Yet by Kyle Simpson — github.com/getify/You-Dont-Know-JS — read alongside this month's work.
Month 2 milestone: Life OS at your domain. Wiki syncs across phone + laptop. Finance tab shows live balance. You've called Claude API from your own code.
03
Mr. Greg v1 — Python + Flask + Claude API
Month 3 · Weeks 9–12
MONTH 3
▼
Now the product: Mr. Greg chat widget — Claude-powered, qualifies ACA leads, NJ ballpark estimates, handoff to Connor. Python + Flask backend. Real URL by end of week 12.
  • Week 9 — Python: Continue MIT textbook. Supplement with MIT OCW 6.0001 free lectures (ocw.mit.edu). Buy Python Crash Course 3rd Ed. by Eric Matthes (~$30 Amazon). Read chapters 1–8 only.
  • Week 10 — Flask: Flask Mega-Tutorial by Grinberg (blog.miguelgrinberg.com, free) — Parts 1–4. Watch CS50W Week 3 (cs50.harvard.edu/web, free) for context.
  • Week 11 — System prompt: Read Anthropic prompt engineering guide. Write Mr. Greg persona — tone, 3 qualifying questions, NJ ACA scenario library, handoff logic. Use Claude to iterate it.
  • Week 12 — Deploy: render.com — free tier, Flask deploys in 10 min. Set API keys in Render dashboard (never in code). Test it. Show Connor.
This is the hard month. Python + Flask + JS + Claude API all at once. Ugly + working beats pretty + broken. Ship first. Polish is month 4.
04
Integrations — HubSpot, Respond.io, Cinch RCS
Months 4–5 · Weeks 13–20
MONTHS 4–5
▼
  • Zapier first (Month 4): zapier.com free plan. Build Zap: handoff button → HubSpot contact. Zero code, 30 minutes. Valid until you understand webhooks better.
  • Respond.io: Free account. Configure Mr. Greg as AI agent inside WhatsApp — same system prompt, no code changes.
  • Cinch RCS (Month 5): cinch.io — read template guide + fallback SMS section before coding. After qualifying conversation → branded card to client's phone. Falls back to SMS automatically.
  • Security: Flask-CORS (1 line), Flask-Limiter (rate limiting). Real Python logging guide (realpython.com, free).
